##  Week-02 Details

### Project: Employee & Department Management API
This module focuses on how backend services communicate over HTTP.  
It includes:

- RESTful API design
- Request Handling with `@RequestMapping`, `@GetMapping`, `@PostMapping`, etc.
- `@PathVariable` and `@RequestBody` usage
- Standard HTTP Status Codes and Responses
- Basic DTO usage

The backend is fully tested using Postman.
